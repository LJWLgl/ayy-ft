### 项目介绍
本项目是基于微信小程序开发的，服务校园二手交易，本系统只包含前端部分，主要包含以下功能：
+ 商品信息发布与浏览
+ 社交（话题、评论）
+ 商品搜索
+ 位置定位
+ 图片上传
+ 手机号绑定
+ 计数系统
+ ...
